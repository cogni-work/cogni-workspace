# Cogni Work

A dark-first, developer-focused theme inspired by modern AI coding assistants. High contrast with bright green accents and monospace typography that signals technical sophistication.

## Color Palette

- **Near-Black**: `#0A0A0A` - Primary dark backgrounds, hero sections
- **Elevated Dark**: `#1A1A1A` - Cards, containers, tertiary backgrounds
- **Muted Blue-Gray**: `#617A91` - Secondary text, metadata, informational states
- **Bright Green**: `#00D084` - CTAs, links, highlights, success indicators
- **White**: `#FFFFFF` - Primary text on dark backgrounds
- **Gray**: `#A0A0A0` - Body text, descriptions on dark backgrounds
- **Dark Gray**: `#666666` - Captions, muted metadata

### Status Colors

- **Success**: `#00D084` - Confirmation, positive metrics (same as accent)
- **Warning**: `#FFB020` - Caution, attention needed
- **Error**: `#FF4757` - Errors, destructive actions, crisis states

## Typography

- **Headers**: JetBrains Mono Bold / fallback: Fira Code Bold
- **Body**: JetBrains Mono Regular / fallback: Fira Code

## Design Principles

1. Dark-first aesthetic: ~70% dark backgrounds (#0A0A0A, #1A1A1A), ~20% muted grays, ~10% vibrant green accents.
2. Green accent is precious: use #00D084 only for CTAs, links, key highlights, and success states — never as a dominant background.
3. Glassmorphism signature: semi-transparent cards (5% white) with subtle white borders (10% opacity) and green glow accents.
4. Monospace everywhere: JetBrains Mono for all text — headers and body — to maintain developer aesthetic.
5. High contrast required: white text (#FFFFFF) on dark backgrounds is the default. Gray text (#A0A0A0) for secondary content.
6. Generous spacing: large padding inside cards (0.25") to create breathing room in the dark layout.
7. Subtle shadows: prefer green glows over heavy drop shadows. Keep shadow opacity low (0.15-0.25).

## Best Used For

Developer tools presentations, AI/ML product pitches, technical architecture reviews, coding platform showcases
